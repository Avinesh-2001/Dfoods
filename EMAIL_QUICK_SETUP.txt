═══════════════════════════════════════════════════════════
          ⚙️  5-MINUTE SETUP PROCESS
═══════════════════════════════════════════════════════════

STEP 1: Create .env file
─────────────────────────
Location: Dfood/backend/.env

Command:
   cd Dfood\backend
   notepad .env


STEP 2: Add email configuration
─────────────────────────────────
Copy-paste this into .env:

   EMAIL_USER=your-gmail@gmail.com
   EMAIL_PASSWORD=your-app-password-here
   ADMIN_EMAIL=abhishek020621@gmail.com


STEP 3: Get Gmail App Password
─────────────────────────────────
1. Visit: https://myaccount.google.com/security
2. Enable: 2-Step Verification
3. Visit: https://myaccount.google.com/apppasswords
4. Generate: App Password for "Mail"
5. Copy: 16-character password (like: abcd efgh ijkl mnop)
6. Paste: In .env as EMAIL_PASSWORD (remove spaces!)


STEP 4: Update .env with credentials
─────────────────────────────────────
Example:
   EMAIL_USER=myemail@gmail.com
   EMAIL_PASSWORD=abcdefghijklmnop
   ADMIN_EMAIL=abhishek020621@gmail.com


STEP 5: Restart backend
─────────────────────────
   cd Dfood\backend
   npm run dev

Look for: "✅ Email server is ready to send messages"
